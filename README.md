# Meu Primeiro PR

Este repositório foi criado para aprender o fluxo de Pull Request.

## O que é um Pull Request?

Um Pull Request (PR) é uma forma de propor mudanças em um projeto.
Você cria uma branch, faz alterações, e pede para alguém revisar antes de juntar ao projeto principal.

## Passos para contribuir

1. Crie uma branch
2. Faça suas alterações
3. Abra um Pull Request
4. Aguarde a revisão
5. Merge!
