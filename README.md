# Meu Primeiro PR

Este repositorio foi criado para aprender o fluxo de Pull Request.

## O que é um Pull Request?

Um Pull Request (PR) é uma forma de propor mudancas em um projeto.
Voce cria uma branch, faz alteracoes, e pede para alguem revisar antes de juntar ao projeto principal.

## Passos para contribuir

1. Crie uma branch
2. Faça suas alteracoes
3. Abra um Pull Request
4. Aguarde a revisao
5. Merge!
